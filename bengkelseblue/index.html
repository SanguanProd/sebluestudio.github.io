Cara Kerja Sistem Ini
Portfolio kamu membaca konten dari file data.json di repo GitHub. Setiap kali kamu mau update konten, cukup export → upload file baru ke GitHub.
1
Edit semua konten di panel-panel sebelah kiri, lalu klik Simpan Draft
2
Klik tombol ⬇ Export data.json di atas — file akan ter-download
3
Buka github.com → masuk ke repo sebluestudio.github.io
4
Klik file data.json → klik ikon pensil ✏️ → klik Upload changes atau drag file baru
5
Klik Commit changes → tunggu 1-2 menit → cek sebluestudio.github.io 🎉
Cara Fix 404 Error
Pastikan repo GitHub-mu sudah benar:
✓
Nama repo harus persis: sebluestudio.github.io
✓
Di Settings → Pages → Source: pilih Deploy from a branch → Branch: main → Folder: / (root)
✓
File index.html harus ada di root repo (bukan di dalam folder)
✓
Folder bengkelseblu/ berisi index.html → untuk akses panel ini
✓
File data.json di root repo → dibaca otomatis oleh portfolio
